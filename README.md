# AI-camera
AI Camera – This Python program transforms your webcam into a highly intelligent vision system using YOLOv8 and Mediapipe Hands. Key Features: Human Detection Uses YOLOv8 to detect people in the camera frame. Draws bounding boxes around detected persons. Hand Tracking & Finger Skeletons Mediapipe Hands detects hand landmarks. Draws hand skeletons (bones) on the video in real time.  Fully compatible with Python 3.10–3.11. Use Case Examples: Gesture recognition (e.g., thumbs up, peace sign). Interactive control with hand gestures. Human activity monitoring with hand movement analysis.
